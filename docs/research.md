# Research: The Evolution of Signal

### Design Philosophy
The website uses a **Dark Industrial** aesthetic to mimic a physical museum after hours. 
- **Typography:** Space Grotesk for a modern, technical feel.
- **Color Theory:** Each era has a "Signaling Color" (Orange for Steam/Telegraph, Blue for Radio/Air, Green for Digital).

### Technical Implementation
1. **Intersection Observer API:** Used in `script.js` to trigger animations as the user scrolls.
2. **CSS Keyframes:** To create "pulsing" signals that represent communication waves.
3. **Dynamic Routing:** A single `exhibit-detail.html` handles multiple items via URL parameters.
